## Описание функциональности

Это учебный проект, который позволяет увидеть работу некоторых алгоритмов. Визуализация реализована на React за счёт Promise и setTimeout. Реализация этого проекта помогла мне разобраться в некоторых алгоритмах и структурах данных. 

## Установка

Клонируйте репозиторий и выполните команду `npm install` или `yarn`, чтобы установить зависимости.

```bash
git clone https://github.com/bvbsis/algososh.git
cd algososh
npm install
```

## Использование

Для запуска проекта выполните команду:

```bash
npm start
```
